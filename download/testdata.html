Test case data

Click here to download the most recent test data set (Version 3.5). This test data set will also work with Version 3.5.1.

This is a zipped .tar file which includes the be.dat for cv_options=5, little_r format observational data, WRF-3/4D-Var ready observational data, first guess and boundary input for 2008020512Z and the AMSU-A and AMSU-B satellite radiance observational data valid at 2008020512Z. This test case data is for training and testing purposes only.

For cv_options=5 training purposes, you can download the pre-generated 1-day forecasts and use the gen_be utility to generate the background error statistics file.

To learn more about WRFDA and how to use it, please see the User's Guide.
