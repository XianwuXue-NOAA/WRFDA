Expectations of Obs Preprocessing WG
• A file or multiple files containing:
– Time, location, parameter type, value
– Instrument type with associated metadata
– Estimated instrument error (representativeness error
is purview of 3DVAR WG)
– Flags from instrument-specific QC
• Instrument-specific QC is handled by Obs
Preprocessing WG
• All files in BUFR format
• Global coverage (issue: huge volumes likely; how long
to keep on-line?)
