Radiance assimilation

Things you should have already done:

  • Compiled the code with BUFR libraries and RTM libraries (CRTM or RTTOV). See the Advanced 3DVAR compile page for details.
  • Downloaded your desired radiance data in BUFR format.

Step 2: Set up your data and namelist

Radiance data files in BUFR format must be linked according to a standard naming convention. You must also set a few namelist variables for each instrument. The following table details the necessary settings for each instrument; mouse over the column headings for a detailed description of each setting.

Instrument File name Read obs rtminit_platform rtminit_satid rtminit_sensor
Advanced Technology Microwave Sounder (ATMS): A microwave observing instrument aboard the NPP-Suomi satellite. atms.bufr use_atmsobs 17 0 19
Atmospheric Infrared Sounder (AIRS): An infrared observing instrument aboard the Aqua satellite. airs.bufr use_airsobs 9 2 11

For more information about Radiative Transfer Models, see the RTM Help Page
