Relevant Links

This page provides links to several sites that may be of interest to WRFDA users.

NCAR GRAPHICS:

If you use graphic tools like RIP4, you may need to acquire NCAR Graphics library.

NCL Example Page:

If you are using NCL (NCAR Graphics Command Language) or would like to learn more about NCL, you may want to visit this page, which provides many examples of ncl scripts which may be used to generate publication quality graphics for WRF and WRFDA files.

NCAR Computational & Information Systems Lab (CISL):

CISL provides support for NCAR's major computing resources (such as Yellowstone) and data archives.

Allocations: If you would like to utilize NCAR's supercomputing resources, you can visit this page to find out your eligibilty and how you might apply for an account.

Free data archives: CISL's Research Data Archive (RDA) provides many meteorological datasets, many of which can be used with WRF and/or WRFDA. You can download these data by registering for a free account.

Unidata Web Site:

Unidata is a division of UCAR which maintains the Network Common Data Form (netCDF) libraries. All WRF input and forecast files, which are used for WRFDA, are in netCDF format.

MMM Web Site:

This is the home page for the Mesoscale and Microscale Meteorology (MMM) Laboratory of NCAR, in which WRF/WRFDA support is based.

NCAR Web Site:

This is the home page for NCAR - the National Center for Atmospheric Research.

UCAR Web Site:

This is the home page for UCAR - the University Corperation for Atmospheric Research, which manages NCAR.
